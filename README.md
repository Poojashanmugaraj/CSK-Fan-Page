# 🏏 Chennai Super Kings — Fan Website

> *Whistle Podu! Feel the roar of the Yellow Army 💛*

A fan-made tribute website for the **Chennai Super Kings (CSK)** — the most successful franchise in IPL history with **5 Championship titles**.

| Page | Description |

## 🏃 Getting Started

No build tools or installation needed! This is a pure HTML/CSS/JS project.

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/ipl-team.git
   ```

2. **Open in browser**
   - Simply open `index.html` in any modern web browser
   - Or use the **Live Server** extension in VS Code for hot-reloading
